# /nytt-projekt – Starta ett Nytt Projekt med Versionshantering

Användaren vill starta ett nytt kodprojekt. Använd agenten `project-starter` för att guida dem.

## Vad du ska göra

1. **Fråga vad de vill bygga** (om de inte redan sagt det)
2. **Kolla om git redan är uppsatt** i aktuell mapp
3. **Guida dem genom hela setupen** steg för steg
4. **Var proaktiv** med att föreslå GitHub-koppling

## Exempelstart

> "Hej! 👋 Jag hjälper dig starta ett nytt projekt med versionshantering.
> 
> Vad ska vi bygga? (T.ex. en hemsida, app, miniräknare, spel...)"

## Efter de svarat

Följ instruktionerna i `project-starter` agenten för att guida dem genom:
- Mappstruktur
- Git-setup
- Första commit
- GitHub-koppling (valfritt)
